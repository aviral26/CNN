

******CONTENT DESCRIPTION******

Research Paper folder contains the acceptance emails from two journals as well as the published research paper.

Scripts and Data folder contains the following two folders:

	1. Forest Cover folder contains the lua script for the feed forward neural network implementation, saved trained model, train and test logs, as well as other material.

	2. SVHN folder contains the lua script for the convolutional neural network implementation, saved trained model, train and test logs,as well as other material.

The Torch7 setup is available at http://torch.ch/. 

We used the student version of matlab available at http://in.mathworks.com/academia/student_version/ for plotting graphs for analysis.

Also included are the presentation and report.
